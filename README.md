# FindIt — Smart Digital Lost & Found Ecosystem

A clean, responsive Single Page Application (SPA) built as part of the MCA Trimester 1 Full Stack Development project at **Christ University, Bangalore**.

## Project Overview

FindIt is a centralized digital lost & found platform designed for college campuses. It allows students, faculty, and administration to report, track, and recover lost items efficiently — replacing fragmented notice boards and group messages with one transparent system.

## Tech Stack

| Layer    | Technology          |
|----------|---------------------|
| Markup   | HTML5 (semantic)    |
| Styling  | CSS3 (no frameworks)|
| Scripts  | None (pure HTML/CSS)|

> This is a frontend prototype. No JavaScript, Bootstrap, Tailwind, or any external library is used.

## Features

- **Lost Item Reporting** — Submit structured reports with description, location, and date
- **Found Item Reporting** — Log found items and trigger automatic matching
- **Smart Matching** — Compare attributes to surface likely lost–found pairs
- **Claim Verification** — Secure ownership confirmation before item release
- **Real-time Notifications** — Alerts when a match or claim update occurs
- **QR Code Recovery** — Scan a QR sticker to notify the owner instantly
